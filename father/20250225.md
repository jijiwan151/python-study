# while

## while
이는 while 다음에 오는 조건문이 참(=True)가 될 때까지 계속 루프가 돌아
```python
a = 1

while a < 10:
    print('a = ', a)

while a < 10:
    print('a = ', a)
    a = a + 1
```

## while 로 메모리 100% 채워서 PC를 폭파 시켜 보기

```python
mem_list = []

while True:
    mem_list.append('이러쿵 저러쿵 어쩌구 저쩌구')

```

# for
## 단순 for 문 돌려보기
```python
for i in range(5):
    print(i)
```

## 단순 for 문 돌려보기 (ragne 범위)
```python
for i in range(1, 5):
    print(i)
```

## list 로 돌려보기
```python
names = ['jiwan', 'youngmi', 'jiyoung']
for value in names:
    print(value)
```

## list 로 돌려보기
```python
names = {'jiwan': 32, 'youngmi': 34, 'jiyoung': 44}
for value in names:
    print(value)
```
